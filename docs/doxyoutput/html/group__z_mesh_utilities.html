This method triangulates the input polygon using ear clipping algorithm.

based on https://www.geometrictools.com/Documentation/TriangulationByEarClipping.pdf & http://abitwise.blogspot.co.uk/2013/09/triangulating-concave-and-convex.html

Parameters
[in]inMesh- input mesh.
[in]faceIndex- face index of the face to be triangulated in the faces container.
[out]numTris- number of triangles in the input polygon.
[out]tris- index array of each triangle associated with the face.
